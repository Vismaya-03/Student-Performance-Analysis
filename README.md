# Student Performance Analysis & Dashboard

## Project Overview
This project focuses on analyzing student academic performance using **Exploratory Data Analysis (EDA)** and building an **interactive Power BI dashboard** to uncover patterns, trends and key factors influencing student outcomes.

The goal is to transform raw educational data into meaningful insights that can support data-driven decision-making for academic institutions.

## Objectives
- Perform data cleaning and preprocessing on real-world messy data  
- Handle missing values and inconsistent categorical entries  
- Conduct Exploratory Data Analysis (EDA) to understand data patterns  
- Identify factors affecting student performance  
- Develop an interactive dashboard for visual storytelling and insights  

## Dataset Description
The dataset contains student-related academic and demographic information, including:
- Student demographics (Age, Gender, Class, Section)
- Academic scores (Math, Science, English)
- Attendance percentage
- Study hours per week
- Assignment submission rate
- Family income
- Internet access availability
- Final academic result

The dataset includes missing values, inconsistencies, and outliers to simulate real-world analytical challenges.

## Data Preprocessing
The following steps were performed:
- Handling missing values using statistical imputation techniques
- Standardizing inconsistent categorical labels
- Detecting and treating outliers
- Data type correction
- Feature understanding and validation

## Exploratory Data Analysis (EDA)
EDA was conducted to uncover relationships and trends using:

- Distribution analysis
- Correlation analysis
- Pairplots and scatter plots
- Performance comparison across categories
- Statistical summaries

### Key Focus Areas
- Attendance vs Academic Performance
- Study Hours vs Scores
- Socio-economic influence on results
- Assignment submission behavior

## Power BI Dashboard
An interactive dashboard was created to present insights visually and enable easy exploration.

### Dashboard Features
- KPI summaries of student performance
- Score distribution analysis
- Attendance impact visualization
- Category-based performance comparison
- Interactive filters and slicers

Dashboard attached in the repository

## 💡 Key Insights

- Exploratory Data Analysis revealed generally weak linear relationships between individual factors and academic performance.
- No single variable strongly determines student outcomes, indicating performance is influenced by multiple combined factors.
- Attendance, study hours and assignment submission show slight positive trends but not strong predictive relationships individually.
- The dataset suggests academic success is multifactorial and may require combined feature analysis or predictive modeling for deeper understanding.
